================================================================================
QUICK START
================================================================================

1. npm install
2. cp .env.example .env
3. Edit .env with your OPENAI_API_KEY
4. OPENAI_API_KEY=sk_... npm run setup
5. Add the output assistant IDs to .env
6. npm run dev
7. Open http://localhost:8888 in browser
8. Test with example-client.html

================================================================================
API ENDPOINTS
================================================================================

POST /.netlify/functions/chat
  Body: { message: string, threadId?: string }
  Response: { success: boolean, threadId: string, response: string }

POST /.netlify/functions/storyteller
  Body: { message: string, threadId?: string, mode: "ai_storyteller" | "st_assist",
          playerCount?: number, sessionLength?: string, scenario?: string }
  Response: { success: boolean, threadId: string, response: string }
